Improving the Running club website

Purpose: Client wanted the website to be improved

Languages used: HTML, CSS

Tools used: WordPress

Timeframe: 01/07/24 - 01/09/24

The website

Task Given: The client asked me to improve the website as it was too difficult to navigate

My improvements:

  • Split the 1 long page into 6 smaller pages to improve navigation issues
  • Created a navigation bar to improve speed of navigation
  • Added CSS to improve accessibility and to retain users

My overview:

I was tasked to improve the Midsummer Fun Run webpage by someone I already knew as they knew I had experience with website development. I enjoyed developing this webpage as I had never used WordPress before so had to learn all the features that it used and how to implement CSS into it. Another part that I had not experienced before, because I had always made projects for myself, was understanding the needs of the client which taught me a lot about communication and feedback.
