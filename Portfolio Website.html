My Portfolio website

Purpose: To show my skills in a visual way

Languages used: HTML, CSS, JavaScript

Tools used: Webstorm

The Code

Lessons learnt:

  • How to write a "visual CV"
  • Using JavaScript to develop visual effects
My overview:

I have a CV which I used to apply to jobs that were not todo with computing but felt that if I were to explain my experience on a piece of paper when applying for a computing job it would be difficult. Therefore, I thought it would be good to develop a website that explained myself and my experience in a visual way. I had also never had a reason to make a website so thought it would be fun to develop one myself and host it so employers could check out my previous projects I had worked on.
